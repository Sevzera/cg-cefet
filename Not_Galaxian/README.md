# not_Galaxian

- Feito em C++ e OpenGL
- Pacotes necessários:
*freeglut3-dev
*libglew-dev
*libsoil-dev

Parâmetros para compilação:
-lGL -lGLEW -lglut -lSOIL

Instruções:
Basta compilar com g++ (usando os parâmetros) e rodar o executável de saída
